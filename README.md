# Cority

A Laravel 12 application built using Laravel Breeze Authentication.

## 🚀 Tech Stack

- Laravel 12
- PHP 8.3
- MySQL
- Laravel Breeze
- Blade
- Tailwind CSS
- Vite

## ✨ Features

- User Authentication
- Login
- Register
- Forgot Password
- Dashboard
- Profile Management
- Responsive Login Page

## 📚 Learning Objectives

This project is being developed to understand Laravel from the ground up, including:

- Routing
- Controllers
- Blade Templates
- Authentication
- Middleware
- Models & Eloquent
- Database Migrations
- CRUD Operations
- REST APIs

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/cority.git

cd cority

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm run dev

php artisan serve
```
